# Note

개별 학습 노트와 학습 로그. 특정 주제를 정리한 글이거나 그날 배운 것의 기록.

## 규칙

- 위치: `wiki/notes/<id>-<slug>.md` 또는 `wiki/notes/YYYY-MM-DD-<slug>.md`
- 첨부 이미지는 `wiki/assets/<note-id>/`에 둔다.
- 실습 코드를 참조하면 `artifact_path`로 `examples/` 경로를 명시한다.
